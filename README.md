## 📊 Project Overview

As part of my journey as a **Data Analyst**, I designed and built a complete interactive Power BI dashboard for a food delivery business — **Omato**.

The goal was to think like a **business stakeholder** — helping decision-makers:
- 📦 Spot top-performing food items
- 🚚 Track delivery efficiency
- 💳 Understand customer payment preferences
- 📈 Monitor monthly transaction trends

---

## ✅ Dashboard Features

### 1. 🔢 Dynamic KPI Cards
- **Total Quantity**, **Total Transactions** & **Average Quantity per Order**
- Update in real-time based on applied filters

### 2. 🎛️ Interactive Filter Panel
- Slicers for **Month**, **Member Type** (Gold/Regular), **Payment Mode**, **Restaurant Type**

### 3. 📅 Total Quantity by Month
- Trend bar chart tracking order volume to spot **peak demand periods**

### 4. 🍽️ Transactions by Food Type & Member Type
- Cross-tab: `Appetizer | Beverage | Breakfast | Dessert | Main Course | Snack`
- Split by **Gold vs Regular** members

### 5. 🥘 Total Quantity by Food Item
- Ranked bar chart — top dishes: Samosa, Butter Chicken, Masala Dosa, Chole Bhature

### 6. 🍩 Transactions by Delivery Status
- Donut chart: **Delivered vs Cancelled** orders

### 7. 💳 Transactions by Payment Method
- **UPI vs COD vs Card** usage split

### 8. 📉 Transactions by Year & Month
- Line chart tracking **month-over-month** growth trends

## ▶️ How to Use

1. Clone this repo
2. Install [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free)
3. Open `Omato_Dashboard.pbix`
4. Link data source to `data/omato_raw_data.xlsx` if prompted
5. Click **Refresh** and explore!
